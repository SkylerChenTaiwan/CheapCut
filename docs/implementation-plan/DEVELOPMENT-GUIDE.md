## 🎯 給 AI 的指示

### 當你接到「開始 Task X.Y」時

1. **檢查依賴**: 查看上面表格的「依賴」欄位
   - 如果依賴的 Task 還沒完成，回報無法開始
   - 如果可以開始，繼續下一步

2. **更新狀態**: 將該 Task 的狀態改為 🟡
   - 填寫「負責人」欄位（例如：Claude A）
   - 填寫「開始」日期
   - 更新「當前正在進行的 Task」section

3. **執行 Task**: 閱讀該 Task 的詳細文件並執行
   - Task 文件位於 `docs/implementation-plan/phase-X-xxx/task-X.Y-xxx.md`
   - 按照文件中的步驟完成

4. **完成後更新**: 完成 Task 時
   - 將狀態改為 ✅
   - 填寫「完成」日期
   - 更新 Phase 完成度（例如：1/6）
   - 更新整體進度（例如：1/45 Tasks (2%)）
   - 移到「近期完成的 Task」section

### 當你接到「我可以並行執行什麼」時

查看「並行規則」section，告訴用戶當前可以並行執行的 Task 組合。

例如：
- Phase 1: Task 1.2 + 1.3 + 1.4 + 1.5 可同時執行（4 人）
- Phase 2 前半: Task 2.2 + 2.5 + 2.11 可同時執行（3 人）

### 當你接到「當前進度如何」時

查看「當前進度」section，報告：
1. 整體完成度（X/45 Tasks）
2. 每個 Phase 的完成度
3. 當前正在進行的 Task
4. 近期完成的 Task
