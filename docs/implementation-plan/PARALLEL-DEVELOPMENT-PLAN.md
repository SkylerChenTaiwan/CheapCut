## 📊 整體時間軸預估

### 單人開發 vs 多人並行

| Phase | 單人開發 | 2 人並行 | 3 人並行 | 5 人並行 |
|-------|---------|---------|---------|---------|
| Phase 0 | 7-11h | 7-11h | 7-11h | 7-11h |
| Phase 1 | 18-25h | 14-18h | 12-15h | 10-13h |
| Phase 2 | 51-66h | 35-45h | 28-38h | 22-30h |
| Phase 3 | 28-41h | 18-25h | 14-20h | 10-15h |
| Phase 4 | 16-21h | 14-18h | 14-18h | 14-18h |
| Phase 5 | 28-34h | 18-22h | 14-18h | 8-12h |
| **總計** | **148-198h** | **106-139h** | **89-120h** | **71-99h** |

### 實際日曆時間（每天 8 小時）

| 開發模式 | 總工時 | 實際天數 |
|---------|--------|---------|
| 單人開發 | 148-198h | 19-25 天 |
| 2 人並行 | 106-139h | 13-17 天 |
| 3 人並行 | 89-120h | 11-15 天 |
| **5 人並行** | **71-99h** | **9-13 天** |
